# Built-in Math Data

This directory contains repo-ready jsonl files for quick TICO-OPD training and evaluation.

## Files

| File | Rows | Use | Source |
| --- | ---: | --- | --- |
| `dapo17k.jsonl` | 17,000 | training | `BytedTsinghua-SIA/DAPO-Math-17k` |
| `dapo17k-00000.jsonl` ... `dapo17k-00003.jsonl` | 17,000 total | optional shards | `BytedTsinghua-SIA/DAPO-Math-17k` |
| `aime24.jsonl` | 30 | evaluation | `OpenRLHF/aime-2024` |
| `aime25.jsonl` | 30 | evaluation | `MathArena/aime_2025` |
| `math500.jsonl` | 500 | evaluation | `HuggingFaceH4/MATH-500` |
| `SOURCES.json` | n/a | provenance | generated by `scripts/prepare_math_data.py` |

All jsonl rows use the normalized keys:

```json
{"prompt": "...", "label": "...", "source": "...", "source_dataset": "...", "row_idx": 0}
```

`dapo17k.jsonl` is a 17,000-row repo-ready subset generated with:

```bash
python3 scripts/prepare_math_data.py \
  --name dapo17k \
  --out-dir data/math \
  --prefer-http \
  --shard-size 5000 \
  --max-rows 17000
```

The upstream Hugging Face split currently reports many more rows than the "17k" name suggests, so the full split is not vendored into Git. Use the same script without `--max-rows` only if you intentionally want to prepare a larger local dataset outside normal GitHub-sized workflows.
